01

When AI Gave Me 100 Hands

Then AI appeared. Not 2 extra hands. But 100 hands. Suddenly I — someone who couldn't write a single JavaScript function — could build a full-stack application. Landing page? Done. API? Got it. Database schema? AI handles it.

It felt like someone who'd been riding a bicycle their whole life suddenly got to ride a rocket.

But a rocket without a control system is just... a flying bomb.

02

Design Chaos: Different Every Time

I started using AI for company projects. Project A: minimalist, sans-serif, 8px radius. Project B — same system, same brand: suddenly AI switched to serif fonts, round buttons, completely different palette. Project C: yet another style entirely.

Every time, a different design. Never consistent.

As a 10+ year Product Manager, this drove me insane. Not because designs were ugly — sometimes each one looked great individually. But put them side by side and it was pure chaos. Business needs consistency. Brand needs a system. Clients looked at 3 pages and thought they were 3 different companies.

AI is brilliant at design. It's just terrible at remembering the rules.

03

The Week AI Deleted Everything

This is the most painful lesson. I will never forget it.

There was a large SaaS project — running in production, needed a full refactor for multi-language, multi-country support. I handed it to AI. It started... fine. Converting this file, translating that string.

Then everything collapsed.

Context got too long, AI started "losing its mind." It fixed one page, broke another. Got Vietnamese working, English disappeared. And the worst part — during one refactor pass, AI deleted nearly all the existing code. An entire week of work. Gone.

I had to redo it. Attempt 1. Attempt 2. Attempt 3. Attempt 4.

Each time, fix one thing, lose another. I sat in front of the screen at 2 AM, staring at endless diff logs, asking myself why I trusted a production system to AI without protection.

I swore: This will never happen again.

4 attempts. An entire week lost. That's when I built the multi-layer protection system.

04

The Lazy Person's Philosophy

I'll be honest: I'm lazy.

Not lazy about thinking — lazy about doing the same thing over and over. Every new project: set up git, configure deploy, create design system, wire i18n, write README... Every time the same. Every time exhausting.

And I realized: Any problem I've faced twice should become an automated process by the third time.

So I started packaging every painful lesson into a reusable skill. Improve it, refine it, make it reproducible. Next time anyone hits the same wall, they just call one command.

A problem encountered twice should never be encountered a third time.
cm-project-bootstrap cm-execution cm-skill-chain
05

Great Products Nobody Knows About

10+ years as a Product Manager at a SaaS company, one obsession haunted me: Self-onboarding.

Amazing product. Tons of features. But new users came in and... got lost. No proper docs. No user guides. No API reference. Support team drowning in repeat questions.

And when I built my own products without a writing team: documentation became torture. Finish coding, completely exhausted, zero energy left to write a README.

So cm-dockit was born — a documentation powerhouse. It reads code, understands code, and auto-generates: API docs, SOPs, user guides, persona analysis, JTBD mapping, process flows. Multi-language. SEO-optimized. Package your product into world-class documentation without writing a single word.

The best product in the world is worthless if nobody knows how to use it.

06

Mission Control for 100 Hands

When you run multiple AI agents, multiple projects, multiple tasks in parallel to maximize productivity... you quickly lose control. Which task is done? Which agent is running? Which deploy went to production? How many tokens have been spent?

I'm a visual person. I need to SEE everything that's happening.

Everything on one screen: pipeline progress, task queue, token tracker, event log. No digging through logs. No running CLI commands. Open it — look — know the situation instantly.

If you can't see it, you can't control it.
