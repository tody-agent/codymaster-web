Đây là quy trình CodyMaster tự động thực hiện khi bạn nhờ nó làm việc.

💬 Bạn Chat — Mô tả ý tưởng

Nói với AI bạn muốn gì, bằng tiếng Việt. Không cần thuật ngữ kỹ thuật.

"Tạo cho tôi 1 trang web giới thiệu spa, có form đặt lịch, giao diện sang trọng"
2

📋 AI Phân tích & Lập kế hoạch

CodyMaster tự động phân tích yêu cầu, hỏi thêm nếu cần, rồi viết ra kế hoạch chi tiết. Bạn duyệt trước khi nó bắt tay vào làm.

→ cm-brainstorm-idea + cm-planning tự động kích hoạt
3

🎨 AI Thiết kế giao diện

Áp dụng 48 quy tắc UX, xây design system phù hợp thương hiệu của bạn. Bạn review bản preview trước khi code.

→ cm-ux-master + cm-ui-preview tạo mockup trực quan
4

💻 AI Code & Test tự động

Viết code theo phương pháp TDD (test trước, code sau). Quét bảo mật. Kiểm tra lỗi. Bạn không cần đọc code.

→ cm-tdd + cm-debugging + cm-secret-shield bảo vệ code
5

🚀 Deploy — Sản phẩm lên mạng

8 cổng kiểm tra an toàn trước khi đưa lên internet. Tự động tạo tài liệu. Rollback nếu có lỗi.

→ cm-safe-deploy → Sản phẩm live trên CDN toàn cầu!

PROMPT MẪU — COPY & DÁN

Bạn nói thế này, AI hiểu ngay

Đây là những prompt thực tế mà người không biết code có thể dùng ngay.

🌐 Tạo Website
"Tạo 1 landing page cho quán cà phê tên Morning Brew. Cần menu, gallery ảnh, và nút gọi đặt bàn."

AI sẽ: Lập kế hoạch → Thiết kế UI sang trọng → Code HTML/CSS/JS → Deploy lên internet.

📊 Xây Dashboard
"Tạo 1 dashboard quản lý doanh thu cho shop bán hàng online. Cần biểu đồ, bảng đơn hàng, và filter theo ngày."

AI sẽ: Phân tích data flow → Xây dashboard với chart → Thêm auth + database.

🐛 Sửa Lỗi
"Trang web của tôi bị lỗi khi bấm nút đặt hàng. Hãy tìm và sửa lỗi cho tôi."

AI sẽ: Phân tích lỗi có hệ thống → Tìm root cause → Viết test → Sửa → Kiểm tra lại.

📱 App Booking
"Tạo 1 hệ thống đặt lịch hẹn cho phòng khám nha khoa. Bệnh nhân chọn bác sĩ, ngày giờ, và nhận email xác nhận."

AI sẽ: Thiết kế UX → Xây booking flow → Tích hợp email → Deploy hoàn chỉnh.

✍️ Content Marketing
"Tạo 1 blog về skincare cho brand mỹ phẩm Serenity. Cần 10 bài viết SEO, trang blog hoàn chỉnh."

AI sẽ: Research keyword → Viết 10 bài → Audit chất lượng → Xây blog đẹp → Deploy.

📝 Tài liệu & SOP
"Đọc toàn bộ code của dự án này và tạo tài liệu kỹ thuật + hướng dẫn sử dụng cho team."

AI sẽ: Scan codebase → Tạo architecture docs → SOP → API reference → User guide.

SAI LẦM THƯỜNG GẶP

Nói thế nào để AI hiểu đúng?

Prompt tốt = kết quả tốt. Đây là những sai lầm phổ biến và cách sửa.

Quá chung chung

❌ SAI

"Tạo 1 website đẹp cho tôi"

✅ ĐÚNG

"Tạo 1 landing page cho studio yoga tên ZenFlow. Tone pastel, tím nhạt + trắng. Cần section giới thiệu, bảng giá 3 gói, và form đăng ký trial."

Muốn tất cả 1 lần

❌ SAI

"Tạo 1 app hoàn chỉnh có mọi thứ: đăng nhập, quản lý, báo cáo, thanh toán, chat..."

✅ ĐÚNG

"Bước 1: Tạo trang đăng nhập + đăng ký. Xong bước này sẽ làm tiếp."

Không review kế hoạch

❌ SAI

AI đưa plan → Bạn: "Oke làm đi" (không đọc)

✅ ĐÚNG

Đọc plan → Góp ý: "Bỏ phần chat, thêm phần gallery vào" → AI sửa → Rồi mới approve.

MẸO SỬ DỤNG

7 mẹo để vibe coding hiệu quả hơn

🎯

Mô tả cụ thể

Nói rõ: tên brand, màu sắc, bao nhiêu section, chức năng gì. Càng cụ thể AI càng làm đúng.

🪜

Làm từng bước

Đừng yêu cầu "tạo app hoàn chỉnh" ngay. Chia nhỏ: trang chủ → form → dashboard → deploy.

👀

Review kế hoạch

AI luôn đưa plan trước khi code. Hãy đọc và góp ý. Đây là lúc bạn "chỉ đạo" quan trọng nhất.

🗣️

Nói tiếng Việt

CodyMaster hiểu tiếng Việt. Bạn không cần dùng tiếng Anh hay thuật ngữ kỹ thuật.

📸

Đưa ảnh tham khảo

🔄

Iterate liên tục

"Sửa nút CTA to hơn", "Đổi font chữ", "Thêm section testimonials". Chat tự nhiên để tinh chỉnh.

💾

Dùng /cm-start

CÂU HỎI THƯỜNG GẶP

Giải đáp thắc mắc

Tôi hoàn toàn không biết code, dùng được không?
Hoàn toàn được! Người tạo ra CodyMaster (Tody Le) cũng không phải developer — anh ấy là Product Manager. CodyMaster được thiết kế để bạn chỉ cần chat bằng ngôn ngữ tự nhiên, AI xử lý toàn bộ phần kỹ thuật.
Vibe coding khác gì ChatGPT copy-paste code?
ChatGPT chỉ đưa đoạn code rời rạc, bạn phải tự ghép, tự test, tự deploy. CodyMaster là một hệ thống hoàn chỉnh: 34 skills tự động chạy — từ lập kế hoạch, thiết kế, code, test, kiểm tra bảo mật, đến deploy. Giống như thuê cả team senior thay vì hỏi 1 người random.
Tôi cần cài gì trước khi bắt đầu?
Bạn cần 1 trong các AI coding agent: Cursor IDE, Claude Code, Gemini Antigravity, hoặc bất kỳ agent nào hỗ trợ skills. Sau đó chạy 1 lệnh cài đặt CodyMaster và bắt đầu chat. Xem hướng dẫn chi tiết ở trang Get Started.
Tốn phí bao nhiêu?
CodyMaster hoàn toàn miễn phí và mã nguồn mở. Bạn chỉ cần trả phí cho AI agent bạn sử dụng (ví dụ: Cursor subscription hoặc Claude API). Nhiều agent có free tier đủ để bắt đầu.
Nếu muốn sửa đổi sau thì sao?
Chat lại! CodyMaster có "bộ nhớ" (cm-continuity) nên nó nhớ context dự án. Bạn chỉ cần nói: "Đổi màu nền thành xanh navy" hoặc "Thêm section testimonials" — AI sẽ cập nhật mà không phá code cũ.
Có thể tạo những loại sản phẩm nào?
Landing pages, blog, e-commerce, dashboard, booking system, SaaS, portfolio, tài liệu kỹ thuật, content marketing... Bất kỳ sản phẩm web nào. CodyMaster thậm chí còn hỗ trợ thiết kế mobile app UI.
