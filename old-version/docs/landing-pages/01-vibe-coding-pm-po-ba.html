SECTION 1: PAIN — "Bạn Có Đang Ở Trong Vòng Xoáy Này?" ​

💡 Hook: Pain Agitation (#9)

Bạn describe ý tưởng rõ ràng. Nhưng kết quả thì...

😤 Scenario 1: "Dev làm không đúng ý" Bạn viết PRD chi tiết, wireframe cẩn thận. Dev build xong — "Ủa, tôi nói khác mà?" Quay lại sửa, mất thêm 1 sprint. Multiply hàng chục feature... bạn đang quản lý hay đang làm translator?

😤 Scenario 2: "Vibe coding = Vibe debugging" Bạn thử dùng AI để tự code. Lần đầu: WOW! Chạy được. Lần 2: bug. Lần 3: spaghetti. Lần 4: dev team nhìn PR với ánh mắt thương hại. "PM code à?"

😤 Scenario 3: "Backlog dài vô tận, ship thì nhỏ giọt" Sprint mới lại plan, lại estimate, lại argue priority. 2 tuần cho 1 API endpoint. Trong khi đối thủ ship daily.

SECTION 10: FAQ — Xử Lý Objections ​

💡 O/CO Framework

❓ "Tôi không biết code, dùng được không?"

✅ Chính xác là dành cho bạn! CodyMaster hoạt động bằng lời describe — bạn nói muốn gì, AI tự TDD, tự test, tự quality gate. Bạn chỉ cần approve kết quả trên Dashboard.

❓ "Code AI viết có production-ready không?"

✅ CodyMaster enforce TDD (viết test trước) + 6-Gate Quality System (static analysis → blind review → security scan → performance check → documentation → ship gate). Code phải pass 6 cổng mới được deploy.

❓ "Khác gì dùng ChatGPT / Cursor / Copilot trực tiếp?"

✅ Những tool đó là "AI viết code". CodyMaster là "Senior Engineer discipline cho AI". Nó thêm TDD, Quality Gate, Working Memory, Judge Agent, Safe Deploy lên trên BẤT KỲ tool nào bạn đang dùng. Không thay thế — mà NÂNG CẤP.

❓ "Team dev sẽ nghĩ gì?"

✅ Dev team sẽ THÍCH. Vì code từ CodyMaster có test, có docs, pass quality gate — giảm 80% thời gian code review. PM tự ship phần non-critical = dev tập trung vào architecture.

❓ "Có phải trả phí không?"

✅ CodyMaster là open-source (MIT License). Bạn chỉ trả phí cho AI provider bạn đang dùng (Gemini, Claude, v.v.) — CodyMaster hoàn toàn miễn phí.

❓ "Setup có phức tạp không?"

✅ 5 phút: npm install -g codymaster → codymaster để cài đặt. Xong. Chạy trên 7 AI platforms, không cần config phức tạp.
