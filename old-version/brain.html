FIVE-TIER MEMORY

Our brain knows what to remember,\nwhat to forget, and what to keep forever.

Inspired by neuroscience: different memories have different lifespans.

Sensory Memory

⏱ Seconds — Not stored

Temporary info within 1 command: "file X has 200 lines." Forgotten immediately after because nobody needs this.

🧠 Like hearing a phone number once — gone instantly

Working Memory

⏱ 7 days — CONTINUITY.md

"Currently fixing auth, goal is deploy, just finished API routes." Remembers what you're thinking and doing.

🧠 Like your grocery list today — needed now

Long-Term Memory

⏱ 30–180 days — learnings.json

"CORS error due to missing proxy config." Lasting lessons — kept only if reinforced ≥ 2 times.

🧠 Like riding a bike — never forgotten

Semantic Memory

⏱ Permanent — qmd / Local Vector

Deep search across massive docs without token limits. Finds concepts and patterns hidden in text.

🧠 Like a photographic reading memory

Structural Memory

⏱ Real-time — CodeGraph / AST

Understands your codebase architecture. 95% token compression by reading AST, not raw code.

🧠 Like an innate sense of spatial awareness
