Quick Start

Two ways to install

Choose the method that works best for you. Both give you the same 50+ skills.

1

Open Codex

Launch Codex in your terminal. If you don't have it yet, install from openai.com/codex.

2

Paste this prompt

Copy and paste this into Codex. It will fetch the installation instructions and set everything up.

codex 
Fetch and follow instructions from https://raw.githubusercontent.com/tody-agent/codymaster/main/.codex/INSTALL.md
3

Start building

That's it! Codex now has 50+ skills. Just describe what you want to build:

example 
// Just talk to Codex normally
Build a landing page for my coffee shop.
Make it look like Stripe's design system.
1

Clone & Build

Clone CodyMaster and build the platform-specific skills.

terminal 
git clone https://github.com/tody-agent/codymaster.git ~/.cody-master
cd ~/.cody-master
npm ci
npm run build:platforms
2

Symlink Skills

Link the skills directory so Codex can find them.

terminal 
# Symlink skills into Codex
ln -s ~/.cody-master/.codex/skills ~/.codex/skills
3

Restart & Verify

Restart Codex. Verify everything works:

terminal 
# Check installation health
cm doctor
📦
Plugin Installation (Optional)

Codex also supports plugins via the marketplace. The CodyMaster plugin is available at plugins/cm/.codex-plugin/. See the plugin manifest for details.

Maintenance

Keeping skills up to date

Skills are generated from the canonical source. Pull updates to get the latest.

1

Pull latest changes

Update your local clone to get the latest skills.

terminal 
cd ~/.cody-master && git pull
2

Rebuild platform skills (if editing source)

If you've edited the source skills, rebuild the platform-specific bundles.

terminal 
npm run build:platforms
