Troubleshooting

Quick fixes for common issues

AI doesn't recognize skill commands
Re-run cm to reinstall. The wizard will detect your tools and place skill files in the correct config directories.
Skills installed but AI ignores them
Start a fresh session. Many AI tools only load skill files at session start. Close and reopen your agent.
Wrong account deployed
Run @cm-identity-guard before every deploy. It verifies GitHub org, Cloudflare account, and project match before touching anything.
AI forgets context between sessions
Ensure CONTINUITY.md exists in your project root. Start each session with: "Read CONTINUITY.md and summarize the current state."
Secret leaked to GitHub
Run @cm-secret-shield before every commit. Also add a pre-commit hook: cm install-hooks sets this up automatically.
Inconsistent UI across pages
Run @cm-design-system at the start of a session to lock design tokens. All UI changes should reference the locked system.
